Вводная:
Есть сервис с endpoint http://api.mathjs.org/v1/
Документация:  http://api.mathjs.org/


Задача:

Используя PyTest написать тесты на сложение, вычитание, умножение, деление и извлечения квадратного корня
Дополнительные требования:
    запросы отправлять с 'User-Agent': 'Python Learning Requests'
    для извлечения квадратного корня в запросе использовать метод GET, для других операций - POST
    использовать параметризованные тесты
    для каждого метода использовать отдельную метку @pytest.mark, для возможности запускать тесты отдельного метода
    используя conftest.py добавить пользовательский аргумент --rounding_index, значение которого будет определять точность округления при проверке результатов (пример добавления пользовательского аргумента http://automated-testing.info/t/perechada-argumenta-iz-komandnoj-stroki-v-pytest/10243)
